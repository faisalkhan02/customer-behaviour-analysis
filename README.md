# 🛒 Customer Behaviour Analysis

An end-to-end data analytics project analyzing customer shopping patterns
to increase revenue, improve retention, and optimize marketing strategies.

## 🔄 Project Workflow

| Phase | Tool | Description |
|-------|------|-------------|
| Phase 1 | Python + Pandas | Data Cleaning & EDA |
| Phase 2 | MySQL (SQL) | 12 Business Insight Queries |
| Phase 3 | Power BI | Interactive Dashboard |

## 📊 Dataset
- **5,000** Customers | **30** Unique Items | **17** Features | **5** Categories
- Total Revenue: **$994.18K** | Avg Spend: **$198.84**

## 📁 Project Files

| File | Description |
|------|-------------|
| `customer_shopping_behavior.csv` | Raw dataset |
| `Customer_Behaviour_Analysis.ipynb` | Phase 1 — Python EDA & Cleaning |
| `Customer_Behaviour_Analysis_Queries.sql` | Phase 2 — 12 SQL Business Queries |
| `Customer_Behavior_Analysis_Dashboard.pdf` | Phase 3 — Power BI Dashboard Export |
| `Customer_Behaviour_Analysis_Report.docx` | Complete Project Report |
| `Customer_Behaviour_Analysis_Presentation.pptx` | Project Presentation |

## 🔍 Key Insights

- 📦 **Electronics** drives 49% of revenue but has the **lowest ratings**
- 👴 **51+ age group** contributes **40%** of total revenue
- 💳 **Subscribers** consistently outspend non-subscribers
- 🌆 **New York** is the top city at $143.9K revenue
- 🍂 **Fall season** is a major revenue dip — only $60K vs ~$240K others
